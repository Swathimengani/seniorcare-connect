A web-based platform to connect elderly patients and families with verified caregivers, nurses, and physiotherapists for home care services.

## Tech Stack
- Frontend: React + Tailwind CSS
- Backend: Node.js + Express.js
- Database: MongoDB Atlas
- Auth: JWT

## Key Features
- User / Caregiver / Admin roles
- Patient profile management
- Booking & scheduling
- Caregiver verification workflow
- Booking tracking & service updates
